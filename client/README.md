# 🌿 AgroPlatform Frontend

Welcome to the **AgroPlatform** frontend, a high-performance, nature-inspired web application designed for modern agricultural commerce.

## 🚀 Key Features

- **Smart Shopping**: Organic product discovery grid with tiered volume discounts.
- **Agriculture Intelligence**: Real-time mandi rates, weather telemetry, and AI pest detection.
- **Seamless Auth**: Google OAuth 2.0 integration and high-security OTP fallbacks.
- **Responsive Design**: Mobile-first architecture with glassmorphism and micro-animations.
- **SEO Optimized**: Advanced meta tagging, structured data, and automated sitemaps.

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite
- **State Management**: Zustand
- **Styling**: Vanilla CSS (Living Forest Design System)
- **Icons**: Lucide React
- **Charts**: Recharts

## 📦 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Setup**:
   Copy `.env.example` to `.env` and configure your API URLs:
   ```env
   VITE_API_URL=http://localhost:5000/api
   ```

3. **Development Mode**:
   ```bash
   npm run dev
   ```

4. **Production Build**:
   ```bash
   npm run build
   ```

## 🔍 SEO & Search Console

This project is optimized for search engines:
- **Sitemap**: Located at `/public/sitemap.xml`
- **Structured Data**: JSON-LD Organization schema in `index.html`.
- **Meta Tags**: Optimized titles and descriptions for high-ranking "AgroPlatform" keywords.
